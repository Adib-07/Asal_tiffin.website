ASAL — Premium Homemade Tiffin Service Landing Page
ASAL (meaning Real/Authentic) is a high-end, responsive landing page designed for a premium homemade tiffin service startup based in Mumbai. The project balances a sophisticated "Obsidian & Gold" luxury aesthetic with the heart and soul of traditional Indian "Ghar Ka Khana" (home-cooked food).

✨ Features
🎨 Immersive UI/UX

Luxury Aesthetic: A modern dark-themed interface using an "Obsidian & Emerald" palette with gold accents.

Custom Interactions: Features a smooth custom cursor follower and grain-texture overlay for a tactile, premium feel.

Scroll Animations: Implements an Intersection Observer-based "Scroll Reveal" system for elegant element entry.

🍱 Dynamic Functionality

Menu Filtering: Interactive menu section allowing users to filter plans by category: Student, Corporate, Health, and Weekend.

Subscription Logic: Dynamic modal that calculates monthly savings (10% discount) versus one-time orders.

Founding Member Waitlist: A functional waitlist system with a real-time counter to build early-stage community engagement.

Review System: A custom-built feedback interface with "Empty State" handling for new businesses, allowing users to submit ratings and testimonials.

🤖 Smart Integration

AI Concierge Interface: A custom-designed chatbot UI specifically prompted to act as a brand-aware assistant (configured for Claude AI integration).

Secure Payments: Integrated with Razorpay checkout (test mode enabled) to handle UPI, Cards, and NetBanking.

🛠️ Tech Stack
HTML5: Semantic structure for SEO and accessibility.

CSS3: Advanced styling using CSS Variables, custom animations (@keyframes), and mobile-first Media Queries.

Vanilla JavaScript: Clean, dependency-free logic for modals, filtering, and API interactions.

Typography: Premium font pairing using Playfair Display (Serif), DM Sans (Sans-serif), and Pinyon Script.
